# 飞书表格预览插件

一个 Chrome 扩展，用来在飞书表格里更清楚地查看当前单元格内容。

适合这些场景：

- 单元格内容太长，在表格里看不全
- 需要对照两个单元格内容
- 需要把 Markdown、JSON、LaTeX、图片或视频链接看得更清楚

## 支持范围

- 浏览器：Chrome
- 页面：飞书表格（Sheets）
- 域名：`*.feishu.cn`、`*.larksuite.com`、`*.larkoffice.com`

## 有哪些能力

- 侧边栏预览当前选中的单元格内容
- 支持单栏、左右分屏、上下分屏三种查看方式
- 支持双分栏对照：一个分栏跟随当前选中单元格，另一个分栏保持冻结
- 不同飞书标签页分别保存自己的布局和预览状态
- 自动识别并渲染这些内容：纯文本、Markdown、JSON、LaTeX、图片链接、视频链接
- 支持手动切换渲染模式：`auto`、`text`、`markdown`、`json`、`latex`、`media`

## 怎么安装

1. 打开 Chrome 的 `chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本项目文件夹：`/Users/vincentliu/Coding/Byte_Work/feishu_sheetmate`
5. 打开任意飞书表格页面
6. 点击扩展图标，打开侧边栏开始使用

支持的页面示例：

- `https://xxx.feishu.cn/sheets/...`
- `https://xxx.larksuite.com/sheets/...`
- `https://xxx.larkoffice.com/sheets/...`

## 怎么使用

1. 在飞书表格中选中一个单元格
2. 打开插件侧边栏查看内容
3. 按需要切换单栏、左右分屏或上下分屏
4. 点击分栏 A 或 B，决定哪个分栏继续跟随当前选中单元格
5. 如果自动识别不符合预期，可手动切换渲染模式

## 局限说明

- 目前只支持 Chrome 中的飞书表格页面，不支持其他浏览器或非表格页面
- 内容提取依赖飞书页面结构；如果飞书改版，可能出现无法正确捕获单元格内容的情况
- 冻结的分栏内容和布局状态只保存在当前浏览器会话中，重启浏览器后不会恢复
- 图片和视频更适合直链或常见飞书媒体链接；部分平台页面只会提供说明和原链接，不会直接在侧边栏内播放
- 如果自动识别结果不准确，通常可以通过手动切换渲染模式解决
